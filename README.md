# Power Platform Credit Approval Workflow

A lightweight M365 solution: SharePoint list as data store, Power Automate approvals, Teams Adaptive Cards notifications, and an optional SPFx dashboard. Includes GitHub Actions for Power Platform ALM.

## Features
- Credit Request intake (SharePoint)
- Approval flow (Power Automate)
- Teams Adaptive Card notification
- Optional SPFx React widget for status
- GitHub Actions CI for solution export/import (Power Platform CLI)
